# Attorney
Short description and motivation.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'attorney'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install attorney
```

## Usage

Install migrations and `attorney.yml` file:
```bash
$ bin/rails generate attorney:install
```
